EN: This program automatically moves the cursor to the center of the screen every 100 seconds. You can change the variable "timer.Interval = [to your value]" to make this procedure either slower or faster. The program is launched via the start.cmd file.

RU: Данная программа каждые 100 секунд автоматически перемещает курсор в центр экрана. Вы можете изменить переменную "timer.Interval = 100" на своё значение чтобы данная процедура происходила либо медленней либо быстрей. Программа запускается через файл start.cmd
